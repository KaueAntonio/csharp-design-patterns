# Factory Method

 - Definir uma interface para criar um objeto, mas deixar as subclasses decidirem que class instanciar.
  - Permite adiar a instanciacao para subclasses.

**Criador**
- `factoryMethod()`: Produto

**Produto**
- `anOperation()`

**Implementação**

* **Criador Concreto**
    - `factoryMethod()`: Produto Concreto

* **Produto Concreto**
    - `anOperation()`
